
## 게임 개요
- 한국에서 얼불춤이라고 불리는 A Dance Of Fire And Ice를 모작하려고 합니다.
- 구현사항
  - 각각의 구체가 다른 구체의 주변을 회전하는 방식
  - 박자가 맞지 않았을 경우 게임오버
  - Input은 현재는 Space를 누르면서 박자를 맞추는 방식으로 진행 예정
  - 스테이지는 3~4개 정도 만들 예정입니다.
